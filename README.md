# 코드 리팩터링 브랜치

설정한 문자 내의 이름을 얻은 후 소스 폴더에서 해당되는 이름의 파일을 목적지 폴더에 자동으로 분류
ex) (아티스트 이름) - 앨범제목_1.트랙이름
   () 문자 입력시 앨범제목 추출하는 방식

* 작동방식
   * 원본폴더 내의 파일 선택
   * 해당 파일에서 () 문자 안에 포함된 아티스트 이름(목표 이름) 추출
   * 목적지 폴더에 존재하는 폴더의 이름 확인
   * 해당 폴더의 이름과 목표 이름이 동일할 경우 파일 전송
